before up docker-compose.yml you need to set your cjdropshipping credentials.

1) CJ_EMAIL=yourEmail,
2) CJ_PASSWORD=yourPassword

also, you can replace CJ_EMAIL and CJ_PASSWORD to CJ_TOKEN (your valid accessToken).
